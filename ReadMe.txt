                   ######################################################
                   ##### Sentiment Analysis on Company Reviews Data #####
                   ######################################################


The data used in this project is the company review data obtained from various job
portals. The data is related to three companies namely - Rio Tinto, Anglo American,
and Glencore. 

The data contains date, location and reviews for a company. 

The tasks performed in this project are listed below -
1. Word Cloud
2. Sentiment Analysis
3. Topic Modelling

This contains some pre processing and cleaning of the text data such as-
1. Removing special characters
2. Removing stop words
3. Converting words to lower case
4. Converting words to their root words
5. Spelling correction
6. Tokenization
